# Estrategia de Cruce de Medias con Python

Este proyecto implementa una estrategia básica de trading basada en:
- Cruce de medias móviles (SMA 20 y SMA 50)
- Descarga automática de datos con yfinance
- Backtesting de capital

Herramientas utilizadas:
- Python
- Pandas
- Matplotlib
- YFinance
